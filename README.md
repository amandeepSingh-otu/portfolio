# Portfolio Website

This is my personal portfolio website showcasing my projects, experience, and technical skills. It features a responsive design with a day/night mode toggle, 3D flip cards, and glassmorphism UI elements.

**Live Site:** [https://amandeepSingh-otu.github.io/portfolio/](https://amandeepSingh-otu.github.io/portfolio/)

## Features

* **Responsive Design:** Fully optimized for different screen sizes.
* **Day/Night Mode:** Includes a toggle to switch between light and dark themes.
* **3D Flip Cards:** Interactive cards to display project details.
* **Glassmorphism UI:** Modern, frosted-glass aesthetic elements.
